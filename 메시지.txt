--- kis_auth.py 파일 수정 방법 ---

1. 'open-trading-api/examples_user/kis_auth.py' 파일을 텍스트 편집기로 엽니다.

2. 파일 내용에서 아래와 비슷한 형태의 기존 코드 한 줄을 찾습니다.

   path = os.path.join(os.path.expanduser('~'), 'KIS', 'config', 'kis_devlp.yaml')

3. 찾은 한 줄을 아래의 새 코드 4줄로 완전히 교체해 주세요.

   # 현재 파일의 위치를 기준으로 프로젝트 루트 경로를 계산
   root_path = os.path.abspath(os.path.join(os.path.dirname(__file__), '..', '..'))
   # 루트 경로와 파일명을 조합하여 전체 경로 생성
   path = os.path.join(root_path, 'kis_devlp.yaml')

4. 'kis_devlp.yaml' 파일 이동:
   마지막으로, 'C:\Users\<사용자이름>\KIS\config' 폴더에 있던 'kis_devlp.yaml' 파일을
   현재 프로젝트의 가장 상위 폴더인 'C:\DigitalTwin\trade_stock'으로 복사하거나 이동해 주세요.
